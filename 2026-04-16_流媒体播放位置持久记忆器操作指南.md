## 项目概述

### 产品定位
ResumePoint是一款专为流媒体重度用户设计的iOS应用,旨在解决流媒体应用播放进度不稳定、需要手动记录时间点的痛点。通过自动检测播放状态和智能记录功能,帮助用户轻松管理多个流媒体平台的观看进度。

### 核心价值主张
- **自动化追踪**: 通过MPNowPlayingInfoCenter自动检测播放信息
- **跨平台管理**: 统一管理多个流媒体平台的观看进度
- **智能提醒**: 在最佳时机提醒用户继续观看
- **极简设计**: 打开即用,无需复杂设置

### 目标用户画像
- **主要用户**: 18-45岁流媒体重度用户
- **特征**: 订阅多个流媒体服务(Netflix、Disney+、HBO Max、Prime Video等)
- **痛点**: 频繁遇到播放进度丢失问题
- **行为**: 每周观看视频内容超过10小时
- **付费意愿**: 中等(痛点明确但替代方案存在)

## 技术可行性分析

### 核心技术方案

#### 方案选择: MPNowPlayingInfoCenter + 混合模式

**关键技术: MPNowPlayingInfoCenter**
- iOS系统提供的媒体播放信息中心
- 可监听第三方应用的播放状态
- 提供以下信息:
  - 标题(title)
  - 艺术家/作者(artist)
  - 专辑/系列(albumTitle)
  - 时长(duration)
  - 播放位置(elapsedPlaybackTime)
  - 播放速率(playbackRate)

**技术限制**:
1. 不是所有流媒体应用都使用MPNowPlayingInfoCenter
2. Netflix、Disney+等可能使用自定义实现
3. 信息准确性取决于应用实现质量

**解决方案: 混合模式**
- **自动检测**: 监听MPNowPlayingInfoCenter(优先)
- **手动编辑**: 用户可修正不准确的标题或时间
- **智能匹配**: 通过API获取完整视频信息

### 技术栈

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| **UI框架** | SwiftUI | 苹果现代化UI框架,开发效率高 |

```swift
import CoreData

@objc(VideoProgress)
public class VideoProgress: NSManagedObject {
    @NSManaged public var id: UUID
    @NSManaged public var title: String
    @NSManaged public var platform: String
    @NSManaged public var currentPosition: Double // 秒
    @NSManaged public var totalDuration: Double // 秒
    @NSManaged public var lastUpdated: Date
    @NSManaged public var isCompleted: Bool
    @NSManaged public var coverImageURL: String?
    @NSManaged public var notes: String?
    @NSManaged public var createdAt: Date
    @NSManaged public var sessions: NSSet? // 关联WatchingSession
}

extension VideoProgress {
    // 计算属性:完成百分比
    var progressPercentage: Double {
        guard totalDuration > 0 else { return 0 }
        return (currentPosition / totalDuration) * 100
    }
    
    // 计算属性:剩余时间
    var remainingTime: Double {
        return totalDuration - currentPosition
    }
    
    // 计算属性:格式化的当前时间
    var formattedCurrentPosition: String {
        return formatTime(currentPosition)
    }
    
    // 计算属性:格式化的总时长
    var formattedTotalDuration: String {
        return formatTime(totalDuration)
    }
    
    private func formatTime(_ seconds: Double) -> String {
        let hours = Int(seconds) / 3600
        let minutes = (Int(seconds) % 3600) / 60
        let secs = Int(seconds) % 60
        
        if hours > 0 {
            return String(format: "%d:%02d:%02d", hours, minutes, secs)
        } else {
            return String(format: "%d:%02d", minutes, secs)
        }
    }
}
```

### 1. 播放状态监听服务

```swift
import Foundation
import MediaPlayer

/// 播放状态监听服务
/// 负责监听MPNowPlayingInfoCenter并解析播放信息
class PlaybackMonitorService: ObservableObject {
    // MARK: - Published Properties
    @Published var currentVideo: VideoInfo?
    @Published var isMonitoring = false
    
    // MARK: - Private Properties
    private let nowPlayingInfoCenter = MPNowPlayingInfoCenter.default()
    private var timer: Timer?
    
    // MARK: - Data Types
    struct VideoInfo {
        let title: String?
        let artist: String?
        let album: String?
        let duration: Double
        let elapsedTime: Double
        let playbackRate: Double
        let timestamp: Date
    }
    
    // MARK: - Initialization
    init() {
        setupObservers()
    }
    
    deinit {
        stopMonitoring()
    }
    
    // MARK: - Public Methods
    func startMonitoring() {
        guard !isMonitoring else { return }
        
        isMonitoring = true
        timer = Timer.scheduledTimer(withTimeInterval: 5.0, repeats: true) { [weak self] _ in
            self?.checkNowPlayingInfo()
        }
        
        // 立即检查一次
        checkNowPlayingInfo()
    }
    
    func stopMonitoring() {
        isMonitoring = false
        timer?.invalidate()
        timer = nil
    }
    
    // MARK: - Private Methods
    private func setupObservers() {
        // 监听应用进入前台
        NotificationCenter.default.addObserver(
            self,
            selector: #selector(appDidEnterForeground),
            name: UIApplication.willEnterForegroundNotification,
            object: nil
        )
        
        // 监听应用进入后台
        NotificationCenter.default.addObserver(
            self,
            selector: #selector(appDidEnterBackground),
            name: UIApplication.didEnterBackgroundNotification,
            object: nil
        )
    }
    
    @objc private func appDidEnterForeground() {
        startMonitoring()
    }
    
    @objc private func appDidEnterBackground() {
        // 在后台继续监听(需要配置Background Modes)
        // 或者停止监听以节省资源
    }
    
    private func checkNowPlayingInfo() {
        guard let nowPlayingInfo = nowPlayingInfoCenter.nowPlayingInfo else {
            currentVideo = nil
            return
        }
        
        // 解析播放信息
        let title = nowPlayingInfo[MPMediaItemPropertyTitle] as? String
        let artist = nowPlayingInfo[MPMediaItemPropertyArtist] as? String
        let album = nowPlayingInfo[MPMediaItemPropertyAlbumTitle] as? String
        
        let duration = nowPlayingInfo[MPMediaItemPropertyPlaybackDuration] as? Double ?? 0
        let elapsedTime = nowPlayingInfo[MPNowPlayingInfoPropertyElapsedPlaybackTime] as? Double ?? 0
        let playbackRate = nowPlayingInfo[MPNowPlayingInfoPropertyPlaybackRate] as? Double ?? 0
        
        // 只有在播放时才记录
        guard playbackRate > 0 else {
            currentVideo = nil
            return
        }
        
        let videoInfo = VideoInfo(
            title: title,
            artist: artist,
            album: album,
            duration: duration,
            elapsedTime: elapsedTime,
            playbackRate: playbackRate,
            timestamp: Date()
        )
        
        DispatchQueue.main.async { [weak self] in
            self?.currentVideo = videoInfo
            self?.handleNewPlaybackInfo(videoInfo)
        }
    }
    
    private func handleNewPlaybackInfo(_ info: VideoInfo) {
        // 这里调用数据存储服务保存记录
        NotificationCenter.default.post(
            name: .didDetectNewPlayback,
            object: nil,
            userInfo: ["videoInfo": info]
        )
    }
}

// MARK: - Notification Names
extension Notification.Name {
    static let didDetectNewPlayback = Notification.Name("didDetectNewPlayback")
}
```

```swift
import Foundation
import CoreData

/// 数据存储服务
/// 负责管理CoreData的增删改查操作
class ProgressStorageService: ObservableObject {
    // MARK: - Published Properties
    @Published var videos: [VideoProgress] = []
    
    // MARK: - Private Properties
    private let persistentContainer: NSPersistentContainer
    private var viewContext: NSManagedObjectContext {
        return persistentContainer.viewContext
    }
    
    // MARK: - Initialization
    init() {
        // 初始化CoreData堆栈
        persistentContainer = NSPersistentContainer(name: "ResumePointDataModel")
        
        // 配置CloudKit同步(可选)
        // persistentContainer.persistentStoreDescriptions.first?.cloudKitContainerOptions = 
        //     NSPersistentCloudKitContainerOptions(containerIdentifier: "iCloud.com.yourapp.ResumePoint")
        
        persistentContainer.loadPersistentStores { description, error in
            if let error = error {
                fatalError("CoreData failed to load: \(error.localizedDescription)")
            }
        }
        
        // 监听播放状态变化
        NotificationCenter.default.addObserver(
            self,
            selector: #selector(handleNewPlayback(_:)),
            name: .didDetectNewPlayback,
            object: nil
        )
        
        // 加载已有数据
        fetchAllVideos()
    }
    
    // MARK: - Public Methods
    
    /// 获取所有视频记录
    func fetchAllVideos() {
        let request: NSFetchRequest<VideoProgress> = VideoProgress.fetchRequest()
        request.sortDescriptors = [NSSortDescriptor(keyPath: \VideoProgress.lastUpdated, ascending: false)]
        
        do {
            videos = try viewContext.fetch(request)
        } catch {
            print("Failed to fetch videos: \(error.localizedDescription)")
        }
    }
    
    /// 添加新的视频记录
    func addVideo(title: String, platform: String, currentPosition: Double, totalDuration: Double) -> VideoProgress {
        let video = VideoProgress(context: viewContext)
        video.id = UUID()
        video.title = title
        video.platform = platform
        video.currentPosition = currentPosition
        video.totalDuration = totalDuration
        video.lastUpdated = Date()
        video.isCompleted = false
        video.createdAt = Date()
        
        saveContext()
        fetchAllVideos()
        
        return video
    }
    
    /// 更新视频进度
    func updateVideo(_ video: VideoProgress, currentPosition: Double) {
        video.currentPosition = currentPosition
        video.lastUpdated = Date()
        
        // 检查是否已完成
        if video.progressPercentage >= 95 {
            video.isCompleted = true
        }
        
        saveContext()
        fetchAllVideos()
    }
    
    /// 删除视频记录
    func deleteVideo(_ video: VideoProgress) {
        viewContext.delete(video)
        saveContext()
        fetchAllVideos()
    }
    
    /// 搜索视频记录
    func searchVideos(query: String) -> [VideoProgress] {
        let request: NSFetchRequest<VideoProgress> = VideoProgress.fetchRequest()
        request.predicate = NSPredicate(format: "title CONTAINS[cd] %@", query)
        request.sortDescriptors = [NSSortDescriptor(keyPath: \VideoProgress.lastUpdated, ascending: false)]
        
        do {
            return try viewContext.fetch(request)
        } catch {
            print("Failed to search videos: \(error.localizedDescription)")
            return []
        }
    }
    
    /// 导出数据为JSON
    func exportToJSON() -> Data? {
        let exportData = videos.map { video -> [String: Any] in
            return [
                "id": video.id.uuidString,
                "title": video.title,
                "platform": video.platform,
                "currentPosition": video.currentPosition,
                "totalDuration": video.totalDuration,
                "lastUpdated": video.lastUpdated.iso8601String,
                "isCompleted": video.isCompleted
            ]
        }
        
        return try? JSONSerialization.data(withJSONObject: exportData, options: .prettyPrinted)
    }
    
    // MARK: - Private Methods
    
    @objc private func handleNewPlayback(_ notification: Notification) {
        guard let userInfo = notification.userInfo,
              let videoInfo = userInfo["videoInfo"] as? PlaybackMonitorService.VideoInfo,
              let title = videoInfo.title,
              !title.isEmpty else {
            return
        }
        
        // 查找是否已存在相同视频
        let existingVideo = videos.first { video in
            video.title == title && video.totalDuration == videoInfo.duration
        }
        
        if let existing = existingVideo {
            // 更新现有记录
            updateVideo(existing, currentPosition: videoInfo.elapsedTime)
        } else {
            // 创建新记录
            _ = addVideo(
                title: title,
                platform: "Unknown", // 后续可通过智能识别确定平台
                currentPosition: videoInfo.elapsedTime,
                totalDuration: videoInfo.duration
            )
        }
    }
    
    private func saveContext() {
        do {
            try viewContext.save()
        } catch {
            print("Failed to save context: \(error.localizedDescription)")
            viewContext.rollback()
        }
    }
}

// MARK: - Date Extension
extension Date {
    var iso8601String: String {
        return ISO8601DateFormatter().string(from: self)
    }
}
```

## UI设计规范

### 设计原则（符合美国市场主流趋势）

#### 1. 极简主义 + 功能性
- **大标题**: SF Pro Display, Bold, 28pt
- **清晰层次**: 主要内容 > 次要信息 > 辅助文本
- **卡片式设计**: 圆角12pt, 微阴影

## 技术难点与解决方案

### 难点1: MPNowPlayingInfoCenter信息不准确

**问题描述**:
- 不是所有流媒体应用都正确实现MPNowPlayingInfoCenter
- 视频标题可能不完整或不准确
- 播放位置可能存在误差

**解决方案**:
1. **混合模式**: 自动检测 + 手动编辑
2. **智能匹配**: 通过API(如OMDb、TMDB)获取完整信息
3. **用户校准**: 允许用户修正不准确的记录
4. **容错机制**: 记录原始数据,支持后续修正

### 难点2: 后台运行限制

**问题描述**:
- iOS对后台运行有严格限制
- 长时间后台监听可能被系统终止
- 电量消耗问题

**解决方案**:
1. **合理使用Background Modes**: 仅申请必要的后台权限
2. **定时唤醒**: 使用BGAppRefreshTask定期更新
3. **智能监听**: 仅在用户使用流媒体应用时激活监听
4. **用户教育**: 在应用中说明后台运行的必要性

### 难点3: iCloud同步冲突

**问题描述**:
- 多设备同时修改同一记录
- 同步延迟导致数据不一致
- 网络问题导致同步失败

**解决方案**:
1. **冲突解决策略**: 采用"最后修改优先"策略
2. **同步状态指示**: 清晰显示同步进度和状态
3. **离线支持**: 本地数据作为主数据源,云同步为辅助
4. **错误处理**: 提供同步失败的重试机制
